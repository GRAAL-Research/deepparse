Dataset Container

To load the data, we use dataset container that are implemented with respect to an interface.

class deepparse.dataset_container.DatasetContainer[source]

Interface for the dataset. This interface define most of the method that the dataset needs to define. If you define another dataset container, the init must define the attribute data.

class deepparse.dataset_container.PickleDatasetContainer(data_path: str)[source]

Pickle dataset container that import a list of address in pickle format. See here for example of pickle address data.
